       ```
       # Modelfile for the opencode agent variant of qwen3:8b.
       #
       # Qwen3:8B scored 0.933 F1 on Docker's 3,570-test tool-calling benchmark.
       # Uses stable Hermes-style tool calling format on Ollama.
       # num_thread 6 to prevent CPU hogging on 12-core ARM64 (Phase 4 finding).
       # NO custom SYSTEM prompt -- opencode provides its own.
       #
       # To create: ollama create qwen3-8b-opencode-agent -f qwen3-8b-opencode-agent.Modelfile

       FROM qwen3:8b
       PARAMETER num_ctx 8192
       PARAMETER num_predict 4096
       PARAMETER temperature 0
       PARAMETER num_batch 1024
       PARAMETER num_thread 6
       ```
    2. Pull qwen3:8b: `ollama pull qwen3:8b`
    3. Create model variant: `ollama create qwen3-8b-opencode-agent -f src/agents/opencode/qwen3-8b-opencode-agent.Modelfile`
    4. Update OPENCODE_MODEL in index.ts to `'qwen3-8b-opencode-agent'`
    5. Update opencode.skill-eval-agent.json: model field to `"ollama/qwen3-8b-opencode-agent"`, models key to `"qwen3-8b-opencode-agent"`

    7. Run 1 warmup + 1 screening trial
    8. If >= 0.85, run 3-trial validation (1 warmup + 3 measured)

    - num_ctx 16384 (more context headroom)
