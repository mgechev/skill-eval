PARAMETER num_ctx 4096

PARAMETER temperature 0
PARAMETER num_batch 1024

Model name: `qwen2.5-3b-skill-eval-agent`
System prompt: 3-line directive + `/no_think`
API: no `think: false` (unsupported by qwen2.5)

### Final Production Benchmark
53.2s avg, reward 1.00, 3 cmds, +/- 0.3s std dev.
5.6x under the 300s target. 4.4x faster than the qwen3.5:4b baseline.

## Decisions
- [Phase 04.1]: qwen2.5:3b is the production agent model -- 3B non-thinking model outperforms all larger/thinking models on superlint_demo
- [Phase 04.1]: qwen3.5:4b-q8_0 is the documented alternative for tasks requiring chain-of-thought reasoning
- [Phase 04.1]: /no_think is kept in system prompt -- qwen2.5 recognizes it and produces more concise responses, boosting reward from 0.97 to 1.00
- [Phase 04.1]: Context pruning adds variance without clear benefit -- retained in code but never triggers with 3-command runs
- [Phase 04.1]: Model size is not correlated with agent quality -- smallest model (3B) beat all larger models
- [Phase 04.1]: qwen2.5:3b shares base blob with grader model -- zero additional disk for agent
